Detailed Description

Perform a "Concerted-Rotation"-like perturbation.

ConRot works by randomly perturbing one torsion in a subchain and then solving IK for the next one.

The allSubchains flag define the comportement of the perturber (default to true). If set to true, the perturber will choose the torsion angle in a randomly chosen subchain. If set to false, the perturber will choose the torsion angle in the subchain defined with the "setSubchainToPerturb" method.

void mma::ConRot::setMaxDChange ( double  maxDChange) [virtual]

Set the maximum allowed change to be applied to a dihedral (angle).

The value of the given argument should always be positive; the method throws a runtime error otherwise. For e.g. if maxDChange = 0.5 and the dihedral to be sampled is alpha, then alpha after the change would be within the interval [alpha - 0.5, alpha + 0.5].
