Detailed Description

A sequential path iterator, allowing for a perfectly regular subdivision of the path.

Subdividing the range [0, path_length] into the minimum number of segments, such that all segments have equal length 'step', and step <= segment_length.

T[0] = 0 , T[1] = 1/p , T[2] = 2/p , ... , T[p-1] = (p-1)/p , T[p] = 1

  • 0 --> initial conformation
  • 1 --> final conformation
  • n/p --> conformation interpolated between the initial and final ones.
