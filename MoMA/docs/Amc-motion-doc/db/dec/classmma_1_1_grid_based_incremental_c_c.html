Detailed Description

This collision checker uses a grid of atoms that enables to test an atom against its neighbors only.

This collision checker is incremental in the sense that each time it is called, it computes the list of atoms that have moved (and updates the grid consequently), and only those atoms are tested for collision. More precisely, two atoms that have not moved are not tested against each other; but an atom that has moved is tested against all its neighbors, including those that have not moved.
