Detailed Description

Perturbs tripeptides based protein by perturbing the particles and generating conformations for the backbones using Inverse Kinematics.

The allParticles flag define the comportement of the perturber (default to true). If set to true, the perturber will perturb a random number of particles (from 1 to numToPerturb). If set to false, the perturber will perturb all the particles defined with the "setParticlesToPerturb" method.

double mma::ParticlePerturber::maxDChange [private]

The maximum absolute random change to be applied to a dihedral (angle).

This change can be positive or negative. For e.g. if maxDChange = 0.5 and the dihedral to be sampled is alpha, then alpha after the change would be within the interval [alpha - 0.5, alpha + 0.5].
