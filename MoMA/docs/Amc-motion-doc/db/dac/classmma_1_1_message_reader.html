Detailed Description

This class establishes a unix domain socket, and automatically accepts all incoming connections and messages.

This is done asynchronously via a secondary thread. All messages are stored in g_message_queue; which is a global and thread-safe object. The messages that have been received can be read via g_message_queue.pop_front() which returns a boost::optional<std::string> object. If there are messages in the queue, pop_front returns the oldest one, and removes it from the queue. If there are no messages in the queue, the object returned is empty.

Exemple usage: #include "MessageReader.hpp"

MessageReader reader("/tmp/mysocket");

// Then later one in the program... #include "MessageQueue.hpp"

boost::optional<std::string> message = g_message_queue.pop_front(); While(message) { std::cout << "Message received: " << *message << "\n"; message = g_message_queue.pop_front(); } // When quitting reader.stop();
