Detailed Description

This collision checker is based on:

  • "Linked-List Cell Molecular Dynamics" (see doc)
  • "Rigid-CLL: avoiding constant-distance computations in cell linked-lists algorithms", V. Ruiz de Angulo, J. Cort√©s, and J. M. Porta, Journal of Computational Chemistry, 2011.

Atoms are distributed within the cells of a grid (an atom belong to only one cell). To detect possible collisions, an atom is checked against the atoms that are located in its own cell or in one of the 26 neighboring cells. Furthermore, atoms are processed by rigid groups.

N.B.: This collision checker is more efficient if the user defines a surrounding box centered around the origin for the molecular system.

Get the atoms in collision in the conformation of a molecular system.

Fill the map 'collAtoms' with the atoms in collision only if 'getAll' is set to 'true'. Every key in the map is an atom and the corresponding value is the set of atoms that are in collision with the 'key' atom. Return 'true' if a collision is found and 'false' otherwise.
