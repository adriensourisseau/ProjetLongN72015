Detailed Description

Performs a Monte Carlo (MC) simulation on a molecular system at a constant temperature.

Example of usage:

MonteCarlo mc; mc.setMaxTrials(10000); mc.setMaxConformations(5000); mc.setTemperature(300); mc.setEnergyComputer(someEnergyComputer); mc.setMolecularSystem(SomeMolecularSystem); mc.setPerturber(someConfGenerator);

mc.run(); shared_ptr<GraphReader> result = mc.getResult();

void mma::MonteCarlo::run ( )

Starts the Monte Carlo Simulation, which will generate a list of new conformations starting from the current conformation of the molecular system.

The simulation will run until either the maximum number of trials or the maximum number of conformations is reached. The molecular system is moved to every generated valid conformation.

Before invoking this method, the molecular system "molSystem" and the energy computer "energyComputer" must be initialized. The method throws a string exception otherwise.

Reimplemented from mma::Planner.
