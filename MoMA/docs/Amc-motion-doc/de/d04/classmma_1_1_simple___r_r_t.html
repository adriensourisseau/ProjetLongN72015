Perform an RRT connection between fromConf and toConf.

Return the last valid conformation along the path (i.e. fromConf if the connection is 'trapped', toConf if 'reached', or one in between if 'advanced') and a boolean specifying whether the connection has succeeded (i.e. 'advanced' or 'reached') or not. Also, the list of atoms in collisions is updated (if getAll is set to 'true').

virtual bool mma::Simple_RRT::getCollisionsOnPath ( LocalPath *  path,
std::set< const psf_amc::amc_Atom * > &  collAtoms,
bool  getAll,
bool  skipFirst = false,
bool  skipLast = false 
) [protected, virtual]

Check the local path for collisions and build the set of involved atoms (if required).

Return true if the path is collision-free and false otherwise.

  • path: local path to be checked
  • collAtoms: list of atoms involved in collisions (filled only if getAll is set to true)
  • getAll: Should all collisions be checked for? If not, stop at the first collision found.
  • skipFirst: Can the first conformation of the path be skipped?
  • skipLast: Can the last conformation of the path be skipped?
