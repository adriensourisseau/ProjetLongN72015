Detailed Description

Encapsulates all the information needed to update the graph after performing an MLTRRT expansion.

Encapsulates all the information needed to construct a new node in the RRT.

MLTRRT variants may need different types of information to add a new node, therefore, the class is left empty and it is up to the deriving classes to decide about the contents.
