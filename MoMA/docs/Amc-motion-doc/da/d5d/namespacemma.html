Generate a file that contains the coordinates of every atom in the given molecular system.

The file begins with an empty line, then the number of atoms on the second line, then the coordinates x, y, z of every atom, one pair of atoms per line.

Sends a CGO to pymol.

The purpose of sendCgo is to provide a limited but simple mechanism of drawing a CGO in pymol (via pipe). A CGO is represented in C++ as a std::vector<CgoItem>.
