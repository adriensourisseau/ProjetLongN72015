Detailed Description

A Manhattan-like Rapidly Exploring Random Tree (ML-RRT).

This tree is built in the continuous conformational space of a molecular system. It works like the simple RRT, except that it uses two types of conformation parameters, labeled as active and passive, and generates their motion in a decoupled manner.

For more, read: "Disassembly path planning for complex articulated objects" by J Cort√©s et. al., Transactions on Robotics 24(2), 2008.
