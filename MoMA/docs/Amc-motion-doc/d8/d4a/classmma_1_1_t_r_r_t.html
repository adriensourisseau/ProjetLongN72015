Detailed Description

The Transition-based Rapidly-exploring Random Tree (T-RRT).

This planner explores the conformational space of a molecular system. It is similar to RRT, except that it accepts/rejects new conformations based on a transition test that steers the exploration toward low-energy regions of the space.

"Sampling-based path planning on configuration-space costmaps", L. Jaillet et al., T-RO, 2010.
