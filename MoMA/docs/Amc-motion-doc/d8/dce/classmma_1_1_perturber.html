Detailed Description

A parent abstract class for all kinds of Perturbers, i.e.

classes which generate a random conformation for a given molecular system by increasing/decreasing (perturbing) the values of the DOFs of a base conformation. Differences between classes relies on the choice of perturbed DOFs (joints).
