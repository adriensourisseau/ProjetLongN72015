Detailed Description

Perturb a random sidechain.

The perturbation or the sidechain then consist in perturbing all the dihedrals in this sidechain.

The allSidechains flag define the comportement of the perturber (default to true). If set to true, the perturber will choose a random sidechain to perturb. If set to false, the perturber will pertub the sidechain defined with the "setSidechainToPerturb" method.

double mma::SidechainPerturber::maxDChange [private]

The maximum absolute random change to be applied to a dihedral (angle).

This change can be positive or negative. For e.g. if maxDChange = 0.5 and the dihedral to be sampled is alpha, then alpha after the change would be within the interval [alpha - 0.5, alpha + 0.5].
