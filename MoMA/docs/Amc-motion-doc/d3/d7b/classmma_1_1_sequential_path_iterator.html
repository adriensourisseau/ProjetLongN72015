Detailed Description

A sequential path iterator.

T[0] = 0 , T[1] = k , T[2] = 2k , ... , T[p-1] = (p-1)k , T[p] = 1

  • 0 --> initial conformation
  • 1 --> final conformation
  • n.k --> conformation interpolated between the initial and final ones.
