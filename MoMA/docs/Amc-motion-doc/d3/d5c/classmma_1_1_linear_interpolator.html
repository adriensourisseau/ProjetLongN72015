Perform the linear interpolation and return the resulting conformation (or NULL if it failed).

The interpolation factor must always be in the range [0,1], otherwise an exception is thrown. If interpolation factor = 0, the initial conformation itself is returned. If interpolation factor = 1, the final conformation itself is returned.
