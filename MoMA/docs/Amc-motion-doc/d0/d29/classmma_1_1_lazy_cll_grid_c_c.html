Detailed Description

This collision checker uses a grid of atoms that enables to test an atom against its neighbors only.

This collision checker is lazy in the sense that it assumes that the atoms that moves are always the same, and does not recompute them. This collision checker is incremental in the sense that each time it is called, it updates the grid (instead of rebuilding it) and checks only the moved atoms for collisions. More precisely, two atoms that have not moved are not tested against each other; but an atom that has moved is tested against all its neighbors, including those that have not moved.
