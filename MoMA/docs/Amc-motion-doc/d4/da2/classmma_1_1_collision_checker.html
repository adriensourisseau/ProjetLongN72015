bool mma::CollisionChecker::checkForCollision ( ) [inline]

Check for collisions in the current state of a molecular system (convenience method).

Return 'true' if a collision is found and 'false' otherwise.

This method is generic and inherited as such by all derived collision checkers On the contrary finally called getCollision method is virtual.

Reimplemented in mma::SimpleCollChecker.

Get the atoms in collision in the conformation of a molecular system.

Fill the map 'collAtoms' with the atoms in collision only if 'getAll' is set to 'true'. Every key in the map is an atom and the corresponding value is the set of atoms that are in collision with the 'key' atom. Return 'true' if a collision is found and 'false' otherwise.

Implemented in mma::GridBasedCLLCC, mma::SimpleCollChecker, and mma::GridBasedIncrementalCC.
