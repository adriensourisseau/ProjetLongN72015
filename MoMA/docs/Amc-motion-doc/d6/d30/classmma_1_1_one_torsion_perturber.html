Member Data Documentation

double mma::OneTorsionPerturber::maxDChange [private]

The maximum absolute random change to be applied to a dihedral (angle).

This change can be positive or negative. For e.g. if maxDChange = 0.5 and the dihedral to be sampled is alpha, then alpha after the change would be within the interval [alpha - 0.5, alpha + 0.5].
