Detailed Description

Abstract parent class for EST-like algorithms. It encapsulates the basic behavior of an EST algorithm which is common to all its variants.

virtual const ESTExtensionData* mma::EST_Template::sampleFrom ( Node *  node) [protected, pure virtual]

This method must be implemented by derived classes.

It should create a new conformation from the node given in argument. This conformation is encapsulated in a ESTExtensionData object (this object is able to store custom information according to the planner's need). If for some reason, the sample process can fail, it should return an ESTExtensionData with a null conformation.

The derived planner is not the owner of the created pointer and should not store it or delete its content.

Parameters:
nodethe node from which the planner is exploring.
virtual bool mma::EST_Template::sampleGoalComponent ( bool  previousValue) const [protected, virtual]

This method allows derived planners to choose when to sample start and goal components in bi-directionnal planners. It should return true if the goal node component should be sampled, or false if the start component should be. By default, the EST_Template alternates between the two components at each iteration.

Parameters:
previousValuePrevious iteration choice.
