Detailed Description

A dichotomic path iterator.

T[0] = 0 , T[1] = 1 , T[2] = 1/2 , T[3] = 1/4 , T[4] = 3/4 , T[5] = 1/8 , ...

  • 0 --> initial conformation
  • 1 --> final conformation
  • k/n --> conformation interpolated between the initial and final ones.
