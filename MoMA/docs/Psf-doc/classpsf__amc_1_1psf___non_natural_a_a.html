Detailed Description

Chemical description of a non-natural amino-acid residue.

Amino-acids residues are basic components of a peptide. Each of them contains a backbone, actually always the same (except protein ends). A sidechain is attached to this backbone, but, unlike natural amino-acids, its structure is totally unknown and must be fully deducted from atomic formula.
