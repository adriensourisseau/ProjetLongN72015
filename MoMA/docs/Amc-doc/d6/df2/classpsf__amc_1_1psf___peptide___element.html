Detailed Description

Chemical description of an amino-acid residue, either natural or non natural.

Intermediate class between generic psf_Residue, dealing with every kind of macromolecule, seen as a chain of elements, like amino-acids or nucleic acids, and more defined classes like NaturalAA and NonNaturalAA.
