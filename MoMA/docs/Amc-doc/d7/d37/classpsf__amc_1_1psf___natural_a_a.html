Detailed Description

Chemical description of a natural amino-acid residue.

Amino-acids residues are basic components of peptides and, then, proteins. Each of them contains a backbone, actually always the same (except protein ends), but which description may vary depending on models. A sidechain is attached to this backbone, from 20 possible kinds.
