Detailed Description

A conformation is mainly a list of values for all joints of related object (from subchain to whole system/multimolecule)

Conformation content : Data, containing a vector of JointData pointers Conformations, a vector of conformation pointers. So, conformation hierarchy follows molecular objects hierarchy.
