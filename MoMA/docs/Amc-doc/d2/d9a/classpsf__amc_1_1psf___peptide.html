Detailed Description

Chemical model of peptide, as in pdb file.

psf_Peptide class derives from psf_Molecule. As a peptide is a combination of chains made of natural amino-acid residues, non natural amino-acid residues plus several groups of atoms, the main data in psf_Peptide object is the list of these subchains.
