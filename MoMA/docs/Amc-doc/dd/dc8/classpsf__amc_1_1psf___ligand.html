Detailed Description

Defines basic, rather small, molecules (ligand).

Unlike proteins, there is no common known architecture, so these are just lists of atoms.

Write generic AMC file for this molecule.

AMC data for ligands is made of one line per joint, each joint described by the list of atoms defining its dihedral.

Parameters:
filename: name of output file
bkb: articulated backbone, useless for ligands
sch: articulated sidechains, useless for ligands
with_header: Include AMC brief documentation header in file
